Weather.AI brings together three end-to-end projects in weather and climate analytics:

Global Climate Data Analytics Dashboard – Interactive climate visualizations, trend analysis, and predictive insights.
Realtime Weather Analysis Pipeline – AWS-powered streaming data pipeline using Lambda, DynamoDB, S3, Snowflake, and Power BI.
SageMaker Weather Prediction – Machine learning models for temperature forecasting using AWS SageMaker and regression techniques.

Together, these projects demonstrate expertise in data engineering, cloud computing, machine learning, and environmental data analytics.
